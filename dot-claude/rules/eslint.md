# ESLint Rules

- `package.json` に `@yasainet/eslint` がある場合は、事前に理解をせよ
- 必要に応じて、`~/Projects/eslint/` を参照して、十分な理解をせよ

## 参照順序 (`@yasainet/eslint` 使用プロジェクトのみ)

1. `~/Projects/eslint/CLAUDE.md` — 設計思想・命名規約
2. `~/Projects/eslint/README.md` — entry points / directory 規約
3. `~/Projects/eslint/src/common/local-plugins/` — 該当 plugin を必要時のみ
